# Icon Placeholders

The icon files (icon16.png, icon48.png, icon128.png) should be created from the SVG logo.

Use an image editor or online tool to convert icon.svg to PNG at these sizes:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

For development, you can use any 16x16, 48x48, and 128x128 PNG images.
